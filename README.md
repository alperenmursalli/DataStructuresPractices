---

## 🚀 Approach

For each problem:

1. Understand the problem deeply
2. Think about possible approaches
3. Identify patterns (HashMap, Two Pointers, etc.)
4. Implement a clean solution
5. Analyze time & space complexity

---

## 🧠 Key Learnings

* Prefer **patterns over memorization**
* Think in terms of **trade-offs (time vs space)**
* Always consider **edge cases**
* Practice explaining solutions clearly

---

## 📌 Notes

* `.class` files are ignored
* Focus is on **clean, readable Java solutions**
* Some problems may include multiple approaches
